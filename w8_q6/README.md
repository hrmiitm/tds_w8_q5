## Visualization Description
This Alluvial Diagram visualizes customer journey flow from awareness to purchase across different digital channels (Social Media, Search Engine, Email, and Direct traffic).

## Data Structure
The visualization represents source-target flow data showing customer movement between touchpoints in the marketing funnel.
