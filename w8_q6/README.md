# Advanced Data Visualization: Alluvial Diagram

## Business Context
This Alluvial Diagram visualizes customer journey flow from awareness to purchase across different channels. The visualization shows how customers move through various touchpoints in their journey: Awareness → Consideration → Interest → Evaluation → Purchase.

## Data Structure
The visualization uses source-target flow data with the following structure:
- **Source**: Origin touchpoint (e.g., Social Media, Search Engine, Direct, Email, or journey stage)
- **Target**: Destination touchpoint or next stage in the customer journey
- **Value**: Number of customers flowing from source to target

## Key Insights
- **Primary Awareness Channels**: Social Media (506) and Search Engine (487) drive the most awareness
- **Conversion Funnel**: The diagram shows clear progression from Awareness → Consideration → Interest → Evaluation → Purchase
- **Abandonment**: Some customers drop off at the Evaluation stage (167 abandoned)
- **Direct Purchase**: Multiple channels lead directly to purchase, with Evaluation being the strongest path (452)

## Tool Used
Created with RAWGraphs (https://rawgraphs.io/) - an open-source data visualization framework for creating publication-ready charts.

## Visualization Type
Alluvial Diagram - ideal for showing flows between categorical stages, perfect for customer journey mapping and conversion funnel analysis.
